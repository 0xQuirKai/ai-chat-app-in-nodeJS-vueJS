# AI Chat Bot
## Description : 
####  AI Chat Bot using cleverbot-free and ExpressJS
 ## USAGE : 
 ### install node modules using :
 ```
 npm install
 ```




### server app.js
```
node app.js
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
