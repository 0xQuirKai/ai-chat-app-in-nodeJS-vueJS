# Covid Tracker
## Description : 
####  Covid Tracker using CovidAPI and build using VueJS in client side and NodeJS 
 ## USAGE : 
 ### install node modules using :
 ```
 npm install
 ```




### server app.js
```
node app.js
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
